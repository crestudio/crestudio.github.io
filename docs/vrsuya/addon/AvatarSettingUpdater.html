Avatar Setting Updater

다운로드 및 설치

1️⃣ VCC에 리스트 등록​

링크를 통해 설치​

아래의 주소를 인터넷 브라우저에서 접속해 주세요

vcc://vpm/addRepo?url=https://server.macchiato.kr/vpm/vpm.json

VCC에서 직접 추가​

  • Settings → Packages → Add Repository → 아래의 JSON 주소 입력 → Add
https://server.macchiato.kr/vpm/vpm.json

2️⃣ 패키지 설치​

원하는 프로젝트에 추가​

  • 추가를 원하는 프로젝트의 Manage Project 버튼을 누릅니다

  • 추가를 원하는 프로그램의 + 버튼을 누릅니다

사용법

3️⃣ VRSuya 제품 Unity Packages를 임포트​

4️⃣ 아바타에 컴포넌트 추가​

원하는 아바타에 컴포넌트 추가​

  • 원하는 아바타에서 Add Component 버튼을 눌러서 VRSuya Avatar Setting Updater를 추가합니다

5️⃣ 옵션 선택 및 적용​

아바타 선택, 설치를 원하는 제품 선택 후 설치​

  • 적용하려는 아바타를 선택 후, 세팅을 원하는 제품을 선택하고, 아바타 업데이트 버튼을 누릅니다

버그 및 관련 문의

Unity의 콘솔 에러 로그가 포함된 전체 스크린샷과 어떤 아바타에서 발생했는지 트위터 @VRSuya 계정으로 DM 부탁드리겠습니다
