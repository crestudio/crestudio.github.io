개발자 업데이트 2024. 07. 25.

2024年7月25日 · 約3分
마끼아또
VRSuya 모델러
아바타 리핑 방지 관련 업데이트 및 밴 웨이브

여름 뮤직 잼 진행 중​

  • 8월 2일까지 접수 중
  • https://hello.vrchat.com/blog/summer-music-jam

아바타 리핑 방지 업데이트​

  • VRChat 게임만 아바타 다운로드 가능하게 변경 (게임 클라이언트 외에는 에셋 파일 취득 불가 개념)
  • 인스턴스에서 아바타 착용한 사람이 있는 경우에만 (단순히 블루프린트로만 에셋 파일 취득 불가)
  • 보안 패치가 안 된 VRC 구 버전은 인스턴스 접속 불가능하게 설정 (아바타 다운로드가 안 되므로)
  • 또한 다른 사용자의 아바타를 리핑한 1900명 이상 계정을 밴(Ban) 처리 함

개발 로드맵 8월 업데이트 예정​

VRC 크리에이터 문서 기여자 가이드 추가​

유니티 AI 네비게이션 지원​

  • 기존에는 월드 업로드 할 때 길 찾기 용 메쉬(NavMesh)를 만들어서 사용할 수 있었으나, 지금은 인게임 속에서 동적으로 변화하는 NavMesh 생성 및 업데이트 가능
  • 장애물이 실시간으로 변화하는 월드에서 몬스터나 NPC들이 플레이어를 추적하는 등 게임 월드 용으로 사용하기 좋을 듯

VRC 제약(Constraint) 오픈 베타​

  • 기존 Unity 제약보다 성능이 좋음
  • SDK를 통해서 자동 컨버팅 가능함

SteamVR 손가락 트래킹 지원​

  • 손가락 트래킹으로 VRC 이용 가능
  • 여러 방법이 있지만 버테탑(Virtual Desktop) 베타를 통해서 사용 가능
  • 다양한 설정 방법은 개발 문서 참조

SteamVR Input 2.0​

  • 거의 대부분 VRChat 기능을 컨트롤러에 바인딩 가능
  • 현재 오픈 베타에서 사용 가능

공식 홈페이지 업데이트​

포털 비주얼 개편​

  • 월드 썸네일이 더 잘 보이도록 이펙트 교체
  • 해당 인스턴스 정보를 가까이서도 읽을 수 있도록 수정
