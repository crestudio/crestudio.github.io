개발자 업데이트 2024.5.2

2024年5月3日
SteamVR 컨트롤러 커스텀마이징 및 핸드트래킹 범용성 확대

일본 골든위크 테마 진행 중(5월 13일까지)​

핸드트래킹 장치 범용성 확대​

  • UI 조작 및 다양한 VRChat 기능에서도 사용 가능해짐

SteamVR 입력 대응 컨트롤러 커스터마이징 확대​

  • 많은 컨트롤러에서 대부분 VRChat 기능을 할당해서 사용 가능해질 것

VRChat 한국어 완전 UI 지원​

  • 다양한 언어 완전 UI 지원
  • 도움말 문서도 번역될 예정

VRCSDK 개발 로드맵 공개​

  • https://ask.vrchat.com/t/vrchat-sdk-roadmap-may-2024/24243

안드로이드 -> iOS 아바타 자동 변환 기능 예정​

월드 및 아바타 갱신 개선 예정​

  • 신규 컨텐츠를 업로드하면 보이지 않는 문제 개선될 듯

VRC Constraint 모든 플랫폼 적용 예정​

  • 올해 말 공개 예정

인스턴스 API​

  • 인스턴스 생성 API를 통해 원하는 인스턴스를 만들도록 바뀔 예정
  • 기존에는 생성 데이터를 담아서 쿼리 보내면 자동 생성되었었음

홈 월드 개편 예정​

  • 홈 월드에 음악 플레이어 있음
  • https://vrchat.com/home/world/wrld_557ebef1-8d35-421e-9e59-959ebaeb4869
