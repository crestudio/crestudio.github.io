개발자 업데이트 2024.11.14

2024年11月14日
프린트(Prints) 기능 추가 / 안드로이드용 빌드 및 테스트 기능

2025년 새해 컨텐츠 모집 중​

  • 포스터, 비디오, 벽에 새겨질 DJ와 같은 로고 같은거
  • 11월 15일부터 모집 시작 / 12월 6일 마감
  • 가이드라인

2024.4.1p2 마이너 패치 릴리즈​

프린트(Prints) 기능 추가 예정​

  • VRC+ 구독자 전용, 구독자는 최대 64개, 비구독자는 최대 20개까지 저장 가능
  • 비구독자는 저장하고 모을 수는 있지만, 만들거나 공유는 불가능
  • 최대 1080p 해상도
  • VRC 사진 찍은 것을 월드에 배치해서 서로 공유하고 볼 수 있는 기능
  • 언제 찍었는지, 누가 찍었는지, 어디에서 찍었는지, 아니면 메시지 같은 메타데이터도 존재

크리에이터스 이코노미 1회성 구매 기능​

  • 1회성 구매는 판매자가 영구적인 구매이거나 월간 구독처럼 선택할 수 있음
  • VRC 메뉴 월드 스토어에서 여러 판매자들의 스토어들을 볼 수 있음

영구 유저 데이터 저장소 사용 사례 모집 중​

크리스마스 연휴 스티커 테마 적용 예정​

안드로이드용 빌드 & 테스트 가능​

  • 직접 안드로이드 기기로 테스트 빌드를 보내는 기능 추가
